This folder contains the logo assets for the Lorton 10K website.

## Required Files

Please add the following logo files to this directory:

1. **lorton-logo.png** - The Lorton 10K Race logo
2. **paramo-logo.png** - The Páramo sponsor logo

- Format: PNG (with transparent background recommended)
- Lorton Logo: Recommended size ~200px width
- Páramo Logo: Recommended size ~180px width
- Both logos should be high resolution for retina displays
